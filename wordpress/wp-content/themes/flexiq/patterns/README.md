# FlexIQ Block Patterns

Pre-designed, drag-and-drop sections for the WordPress Block Editor (Gutenberg).

## How to Use Patterns

### Method 1: Pattern Inserter (Recommended)

1. **Open Block Editor** - Edit any page/post
2. **Click the `+` button** (Add block)
3. **Go to "Patterns" tab**
4. **Find "FlexIQ Patterns" category**
5. **Click a pattern** to insert it
6. **Customize** the content

### Method 2: Slash Command

1. Type `/` in the editor
2. Start typing pattern name (e.g., `/hero`)
3. Select from suggestions
4. Press Enter to insert

### Method 3: Browse All Patterns

1. Click `+` button
2. Click "Explore" at the top
3. Filter by "FlexIQ Patterns" category
4. Preview and insert
